# ☁️ Introdução ao Microsoft Azure

![Azure](https://docs.microsoft.com/en-us/media/logos/azure.svg)

O **Microsoft Azure** é uma plataforma de **cloud computing** que oferece serviços como VMs, bancos de dados, inteligência artificial, análise de dados e muito mais.

---

## 🚀 Vantagens do Azure

- Integração nativa com produtos Microsoft  
- Alta disponibilidade e escalabilidade  
- Segurança e compliance robustos  
- Suporte a múltiplas linguagens e frameworks  
- Ferramentas de monitoramento avançadas

---

## 💡 Serviços comuns

| Serviço       | Função                       |
|---------------|------------------------------|
| Virtual Machines | Máquinas virtuais na nuvem  |
| Azure Blob Storage | Armazenamento de objetos  |
| Azure SQL Database | Banco de dados gerenciado |
| Azure Functions | Computação serverless        |
| Azure VNet       | Redes privadas na nuvem     |
